# Domain Docs

Before exploring the codebase:

- Read the root `CONTEXT.md` when it exists.
- Read relevant decisions under `docs/adr/`.
- Proceed silently when these files do not exist.
- Use terminology defined in `CONTEXT.md`.
- Explicitly flag proposals that conflict with an existing ADR.
